    - When "Same role" bucket exists, includes role change frequency sentence
    - Always includes top transition highlight sentence
    - Includes seniority trend when seniority data is available (using parseSeniorityLevel on sourceRole vs destinationRole)
    - Returns 2-3 sentences joined as a single string
    - Sentences are ordered by "notability" score (most interesting pattern first)

    computeInsights (orchestrator):
    - Returns InsightsData with all three sections populated
    - Handles empty migrations gracefully (empty topDestinations, empty roleBuckets, empty patternSummary)
    - Handles very small result sets (less than 3 migrations) with reduced/caveat output
  </behavior>
  <action>
    Add to src/lib/insights.ts:

    Function: generatePatternSummary(topDestinations: TopDestination[], roleBuckets: RoleBucket[], migrations: MigrationRecord[], searchedRole: string): string
    - Detect 4 pattern types, score each by "notability":
      1. Concentration: score = top destination percentage (if >= CONCENTRATION_THRESHOLD). Template: "{percentage}% of leavers went to {company}, making it the dominant destination."
      2. Role change: score based on "Same role" bucket size (interesting when very high >60% or very low <20%). Template varies: "Only {pct}% kept a similar title..." or "{pct}% stayed in a similar role..."
      3. Top transition: always included, score = count of most common dest role. Template: "The most common move is to {role} ({pct}%)."
      4. Seniority trend: use parseSeniorityLevel on sourceRole vs destinationRole, average delta. Template: "Most transitions were to {similar/more senior/less senior} positions."
    - Sort by score descending, take top 2-3 patterns
    - Join sentences with space into natural paragraph
    - Return empty string for empty migrations

    - Guard: if empty migrations, return { topDestinations: [], roleBuckets: [], patternSummary: "", totalMigrations: 0 }
    - Compute totalMigrations = sum of all counts
    - Call computeTopDestinations, computeRoleBuckets, generatePatternSummary
    - Return assembled InsightsData
