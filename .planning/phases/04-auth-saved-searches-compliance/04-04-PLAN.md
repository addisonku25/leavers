  <name>Task 1: Saved searches server actions</name>
  <files>src/actions/saved-searches.ts</files>
  <action>
1. Create `src/actions/saved-searches.ts` ("use server"):
   - Import: `auth` from "@/lib/auth", `db` from "@/lib/db", `savedSearches` and `searches` from "@/lib/db/schema", `headers` from "next/headers", `nanoid` from "nanoid", `eq` and `and` from "drizzle-orm"

   - `saveSearch(searchId: string)`:
     - Get session via `auth.api.getSession({ headers: await headers() })`
     - If no session: return `{ error: "unauthorized" }`
     - Check if already saved (same userId + searchId) -- if so, return `{ success: true }` (idempotent)
     - Insert into savedSearches: id (nanoid), userId (session.user.id), searchId, createdAt (new Date())
     - Return `{ success: true }`

   - `deleteSavedSearch(savedSearchId: string)`:
     - Get session, verify auth
     - Delete from savedSearches WHERE id = savedSearchId AND userId = session.user.id (ownership check)
     - Return `{ success: true }`

   - `getSavedSearches()`:
     - Get session, verify auth
     - Query savedSearches joined with searches table to get company + role info
     - Return array of `{ id, searchId, company, role, createdAt }`
     - Order by createdAt desc (newest first)

   - `isSearchSaved(searchId: string)`:
     - Get session -- if no session, return `{ saved: false }`
     - Query savedSearches WHERE userId = session.user.id AND searchId = searchId
     - Return `{ saved: boolean }`
  </action>
