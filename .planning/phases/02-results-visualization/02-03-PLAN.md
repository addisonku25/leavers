  <action>
    **Install dependencies:**
    ```bash
    npm install d3-sankey d3-shape && npm install -D @types/d3-sankey @types/d3-shape
    ```

    **sankey-diagram.tsx** ("use client"):
    - Props: `{ data: SankeyData; width: number; height: number }`
    - Use d3-sankey layout engine for node positioning and link path generation. Use React JSX for all SVG rendering (NOT D3 DOM manipulation -- per research anti-pattern warning).
    - Layout: `sankey().nodeWidth(15).nodePadding(10).nodeAlign(sankeyJustify).extent([[margin, margin], [width - margin, height - margin]])`
    - Node rendering: `<rect>` elements colored by category (source = blue-ish, company = slate, destination = emerald-ish -- use Tailwind CSS variable colors where possible via hsl())
    - Node labels: `<text>` elements positioned to the right of source nodes, left of destination nodes, and centered above company nodes. Truncate long names with ellipsis.
    - Link rendering: `<path>` elements using `sankeyLinkHorizontal()` from d3-sankey. Default opacity ~0.3, stroke color matching source node.
    - Hover interaction: Track `hoveredLink` in state (index or null). On link mouseEnter, set hoveredLink. On mouseLeave, clear it. When hoveredLink is set:
      - Highlighted link: opacity 0.7
      - Connected links (same source or same target): opacity 0.5
      - Unrelated links: opacity 0.1
      - Show tooltip div (absolute positioned over SVG container) with: company name, role, count
    - Use CSS transitions for opacity changes (transition-opacity duration-200) to avoid re-render jank (per research pitfall #4).
    - Guard: if data.nodes.length === 0, render nothing (return null). The "always show Sankey" decision means the parent decides whether to render; the component itself just guards against empty data.

    **Wrap in responsive container inside the component:**
    - Use ResizeObserver pattern from research: useRef for container div, useState for measured width, useEffect with ResizeObserver
    - Fixed height of 400px, width from container measurement
    - Only render SVG once width > 0

    **sankey-error-boundary.tsx** ("use client"):
    - Class component error boundary (per research code example)
    - Props: `{ children: ReactNode; fallback: ReactNode }`
    - On error: render fallback
    - Fallback component: a subtle note like "Flow visualization unavailable" in text-muted-foreground with a small icon (per locked decision: graceful degradation, never block the whole page)
